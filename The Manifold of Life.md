## Scale 1 — The Cell

The cell is the fundamental unit of life. And it is also the clearest demonstration of manifold geometry in biology.

### The Cellular State Space

```
WHAT A CELL'S STATE SPACE CONTAINS
════════════════════════════════════════════════════════════════

  Every cell exists in a configuration of:

  Gene expression    →  which genes are active / silent
  Protein levels     →  what's being built inside the cell
  Membrane potential →  electrical state of the cell boundary
  Metabolic rate     →  how fast the cell is processing energy
  Structural shape   →  the physical geometry of the cell itself

  Every unique combination of these = one point in state space.

  The full set of all possible combinations = the cellular manifold.

  A human cell has roughly 20,000 genes.
  Each gene can be on or off or partial.
  The state space is unimaginably large.

  But the cell doesn't wander it randomly.
  It falls into basins.
  And stays there.
```
